## Custom Slash Commands

All custom build commands live in `.claude/commands/`. Run them with `/phase10`, `/phase11`, etc.

### Build Phases (Lead Management + Attribution)
- `/phase10` — Lead Management Database & Backend (leads table, activities, scoring, conversion)
- `/phase11` — Lead Management UI (Kanban, create form, detail page, conversion modal)
- `/phase12` — Attribution Engine Backend (touchpoints, channel analytics views, triggers)
- `/phase13` — Attribution Dashboard UI (channel performance, lead sources, customer journeys)
- `/phase14` — Lead + Attribution Testing & Hardening (error boundaries, 20+ tests, security)

### Utility Commands
- `/catchup` — Summarize recent changes
- `/deploy-check` — Pre-deployment verification
- `/pr` — Create pull request
- `/debug` — Debug issues
- `/refactor` — Refactor code
- `/architecture` — Architecture review
- `/spec` — Write feature specs
- `/resource` — Generate new CRM resource
- `/meeting-prep` — Prep for sales meetings
- `/deal-review` — Review deal pipeline
- `/email-sequence` — Generate email sequences
- `/battle-card` — Create competitive battle cards
