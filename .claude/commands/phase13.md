# Phase 13: Attribution Dashboard — UI Components

Read CLAUDE.md for project context. This phase builds the attribution and lead analytics frontend.

## Attribution Dashboard Page

Create `src/attribution/` directory with these components:

### AttributionDashboard.tsx
- New top-level page at `/attribution`
- Tab layout with 3 tabs: **Channel Performance**, **Lead Sources**, **Customer Journeys**
- Date range picker at top (last 7 / 30 / 90 days / custom)
- Wrap in CrmErrorBoundary

### ChannelPerformance.tsx (Tab 1)
- Fetches from `channel_attribution_summary` view
- **Bar chart**: leads generated by channel (horizontal bars, sorted desc)
- **Revenue comparison cards**: First-touch revenue vs Last-touch revenue per channel
- **Table below chart** with columns: Channel, Source, Leads Generated, Contacts Touched, Deals Influenced, First Touch Leads, Last Touch Deals, First Touch Revenue, Last Touch Revenue, Total Touchpoints
- Color-code channels consistently: organic_search=green, paid_search=blue, social_organic=purple, social_paid=pink, email=orange, referral=teal, direct=gray, display=yellow, affiliate=indigo, offline=brown

### LeadSourceAnalytics.tsx (Tab 2)
- Fetches from `lead_source_performance` view
- **Funnel visualization**: Total Leads → Qualified → Converted (with conversion rates between stages)
- **Table**: Source, UTM Source, UTM Medium, UTM Campaign, Total Leads, Qualified Leads, Converted Leads, Conversion Rate %, Avg Lead Score, Avg Days to Convert
- Sort by total_leads DESC default
- Highlight rows where conversion_rate > 20% in green
- Filter by source dropdown

### CustomerJourneyTimeline.tsx (Tab 3)
- Fetches from `customer_journeys` view
- **List of customer journey cards**, each showing:
  - Person name + email
  - Lead source badge
  - Timeline bar: lead_created → converted → deal_created → current deal stage
  - Total touchpoints count
  - Days in funnel
  - Deal amount (if exists)
- Click a card to expand and show full touchpoint timeline
- Filter by: deal stage, lead source, date range
- Sort by: days_in_funnel, deal_amount, total_touchpoints

### TouchpointTimeline.tsx (Reusable Sub-Component)
- Used inside CustomerJourneyTimeline cards AND inside LeadShow.tsx
- Vertical timeline showing each touchpoint chronologically
- Each touchpoint node shows:
  - Icon by touchpoint_type (same icon mapping as lead activities)
  - Channel badge (color-coded)
  - Source + campaign if present
  - Page URL (truncated, with tooltip for full URL)
  - Timestamp
  - Attribution flags as small badges: 🏁 First Touch, 🎯 Last Touch, ⭐ Lead Creation, 💰 Deal Creation

### AttributionModelToggle.tsx (Future-Ready)
- Simple toggle between "First Touch" and "Last Touch" attribution display
- When toggled, ChannelPerformance highlights the selected model's revenue column
- Placeholder for future: "Linear", "Time Decay", "U-Shaped" models (disabled with tooltip "Coming soon")
